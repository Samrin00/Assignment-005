                  ANSWER OF THE QUESTIONS

1.  What is the difference between var, let, and const?
   They are used to create variables in JavaScript.
   var – Old way to create variables.
   let – Modern way, and you can change the value later.
   const – Used when the value should not change.




 2. What is the spread operator ...?
    The spread operator is used to expand or copy items from an array or object.
    It copies the old array and adds new values.




3. What is the difference between map(), filter(), and forEach()?
  They are used to work with arrays.

 map()
  Creates a new array after changing each value.
  const doubled = numbers.map(num => num * 2);

ilter()
  Creates a new array with some values removed.
  const even = numbers.filter(num => num % 2 === 0);

forEach()
 Just runs a function for each item.
 It does not create a new array.





4. What is an arrow function?
  An arrow function is a short way to write a function.
  It is shorter and easier to write.




5. What are template literals?
  Template literals are used to insert variables inside a string easily.
   They use backticks instead of quotes.
   ${} is used to insert variables or expressions.
